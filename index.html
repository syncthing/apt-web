The release channel is updated with full release builds, approximately once a week. This is what most people will want to run.

# Add the release PGP keys:
curl -s https://syncthing.net/release-key.txt | sudo apt-key add -

# Add the "release" channel to your APT sources:
echo deb https://apt.syncthing.net/ syncthing release | sudo tee /etc/apt/sources.list.d/syncthing-release.list

# Update and install syncthing:
sudo apt-get update
sudo apt-get install syncthing

Development Builds

The devel channel contains the latest development build. This is bleeding edge stuff, not guaranteed to work.

# As above, except use the "devel" channel instead of "release":
echo deb https://apt.syncthing.net/ syncthing devel | sudo tee /etc/apt/sources.list.d/syncthing-release.list
